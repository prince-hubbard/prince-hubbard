# 💫 About Me:
I’m Ezra Ogidan, a full-stack developer and founder of hospitallia.africa under ClaudesecTech.<br>I build tools that solve real problems for hotels, restaurants, and service businesses across Africa.<br><br>I enjoy turning messy business processes into clean, fast, automated systems.<br>Most of my work sits around Django, Python, Tailwind, and modern backend engineering.<br>I’m currently focused on scaling Hospitallia into a smarter, data-driven hospitality platform.<br><br>If I’m not improving a product, I’m learning, testing new ideas, or refining features that help businesses work better.<br><br>Always open to collaboration, feedback, and interesting tech conversations.
